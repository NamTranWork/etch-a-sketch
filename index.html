Nam's Etch-A-Sketch

INSTRUCTIONS

Use the RGB sliders to create a custom color. As you move the sliders, the color in the small box will update to show your selection. When you hover or click on the grid (depending on the current mode), each square will be colored with your selected color.

You can toggle between Mousedown Mode and Hover Mode using the button labeled "Mousedown Mode". You can also switch to the eraser tool to clear squares. Adjust the grid size any time with the "Set Grid Size" button.
